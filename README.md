# lamp_script

使用方法：  
1.下载脚本并增加脚本执行权限：  
```
wget https://raw.githubusercontent.com/suifengpiaoyang/lamp_script/master/lamp.py && chmod +x *.py 
```
2.执行脚本：  
`./lamp.py`

3.输入 1 ,按 Enter 。  
然后，输入你想设置的数据库的新密码（**图片中的密码是123456，只为了展示而用，请实际使用强密码**）

4.结束。  
检验页面最好到 chrome 浏览器，因为我发现，一些浏览器需要有域名才能正常显示。
